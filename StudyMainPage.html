Definitions

This workbook contains ARC study data that was developed from financial and other information sources. Data is analyzed for a calendar year which is typically the year before the study is published.
Base Year is the data year. Historical Base Year data is included when ARC has previously researched this topic.
Forecasts are provided for five years beyond the Base Year.
Revenues are in millions for all currencies.
Units are in thousands of products.

   
ARC Study Data Sheets

This workbook contains several ARC datasets:

  • Base Year: This is the basis for competitive analysis about market leading suppliers for this study topic.
  • Trend: This forms the basis for trend analysis of market segments. It always contains forecasts, and also includes historical data where ARC has previously researched this topic.
Data is provided in two forms:
  • DB: This form provides both USD and local currency data columns. It is best for use in pivot tables or loading into other tools that require a database-like structure. There is one data point per line.
  • Local: This form, in your selected currency, is compact, easy to visualize, and best suitable for developing your own charts, tables, reports using Excel formulas.
All datasheets may contain some data that is counts of products. these are all identified in the "Units" columns in the data sheets.
   
Visualization Sheets

This workbook also contains sheets that show competitive and trend information from various perspectives to help assess the value of the information.

Currency Selection
Select the local currency that you want to use throughout ths workbook. Currency selections change each year and affect the apparent growth rates.
Competition
Select market segments and vew the relative positions of product suppliers by associated revenue.
Trends
Select market segments and view trends in charts and tables.
CAGRs
Select market segments and view historical and forecasted growth.
Trend Stats
Select market segments and view statistics relative to straight line growth (least squares fit).



Currency Conversion Tables

This workbook contains a currency conversion sheet, "Currency Selection". This sheet includes a table created by ARC from publically available data. This table is used to convert ARC data in US dollars ("USD") to your local currency.
